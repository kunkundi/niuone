# README 展示素材

本目录存放仓库主页使用的展示素材：

更新或重新录制动图时，参见 [README GIF 更新流程](GIF_WORKFLOW.md)。

- `capital-flow.gif`：A 股板块与活跃股票、行业主力净流入 / 净流出动画；
- `market-breadth.gif`：沿日内曲线悬停查看涨跌停、炸板、红绿盘家数和量能指标；
- `practice-trading.gif`：每日 / 累计收益切换与交易日历；
- `market-monitor.gif`：展开并滚动查看盘后监控详情；
- `twitter-monitor.gif`：展开文本与图文推文，并打开图片预览；
- `us-ratings.gif`：展开个股机构评级详情，并切换历史日期；
- `dashboard-settings.gif`：浏览配置分组、切换交易策略并打开行情与资金流设置。

七个 GIF 均使用暗色主题并以 1200 像素宽输出；市场情绪、推特监控和美股评级使用 1200 × 675 的 16:9 视野，其余动图使用 1200 × 750 完整页面视野。素材包含鼠标移动、点击反馈或滚动过程，来自公开演示页或隔离的本地临时运行目录，不包含管理员密钥、模型密钥、Webhook、数据库或其他私有运行数据。
